[Linux] NIS

By Heejoon Lee
Posted 2022-08-31 Updated 2022-09-01 1 min read

NIS(Network Information Service)

NIS allows a single server to provide login information to other machines.

Setup

Server

  1. Required daemons:

    DaemonDescription
    ypservMain NIS server daemon
    yppasswddSet password
    ypxfrdImprove mapping performance

  2. Start and enable daemons:

    1
2

     $ systemctl start ypserv yppasswdd ypxfrd
 $ systemctl enable ypserv yppasswdd ypxfrd

  3. Set NIS domain name:

    • Use nisdomainname:

      1

        $ nisdomainname testdomain.co.kr

    • Add NISDOMAIN to /etc/sysconfig/network:

      1
2

        $ vi /etc/sysconfig/network
  NISDOMAIN=testdomain.co.kr

  4. Refresh info

    Whenever the NIS information is changed(a new user is added, …), recompilation is required.

    1

     $ cd /var/yp ; make

    Example output:

    1
2
3
4
5
6

     $ cd /var/yp ; make
 $ ls
 Makefile testdomain.co.kr
 $ ls testdomain.co.kr
 group.bygid     mail.aliases    protocols.byname    services.byname
 hosts.byaddr    passwd.byname

Client

  1. Required packages:

    PackageDescription
    ypbindMain NIS client daemon
    yp-toolsTools for configuring NIS

  2. Set NIS domain:

    • Use nisdomainname:

      1

        $ nisdomainname testdomain.co.kr

    • Add NISDOMAIN to /etc/sysconfig/network

  3. Set NIS server and domain in config file /etc/yp.conf:

    1
2
3
4

     $ vi /etc/yp.conf
 server nis.testdomain.co.kr
 ypserver nis.testdomain.co.kr
 domain testdomain.co.kr

  4. Start daemons

    1
2

     $ systemctl start ypbind
 $ systemctl enable ypbind

Commands

CommandDescriptionUsage
nisdomainnameShow/set NIS domain name$ nisdomainname
→ Show NIS domain
$ nisdomainname test.or.kr
→ Set NIS domain to test.or.kr
ypwhichNIS client. Show server name, map file names$ ypwhich
→ Show NIS server name
$ ypwhich -m
→ Show NIS map file names
ypcatNIS client. Check the content of NIS map files$ ypcat hosts.byname
→ Print hosts.byname NIS map file content
yptestNIS client. Check NIS settings$ yptest
yppasswdNIS client. Change password of a user$ yppasswd heejoon
→ Change the password of an NIS user heejoon
ypchshNIS client. Change the shell of a user$ ypchsh heejoon
→ Change the default shell of an NIS user heejoon
ypchfnNIS client. Change the info of a user$ ypfn heejoon
→ Change the information of an NIS user heejoon
